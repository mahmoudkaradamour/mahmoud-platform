# UI System & Design Language

## 1. Google UX & Material 3
We adhere strictly to **Material 3 (M3)** design principles while customizing for our brand identity.

### Core Principles
*   **Zero Cognitive Load**: Use consistent icons, clear labels, and meaningful animations.
*   **Micro-interactions**: Subtle feedback for actions (e.g., button press, swipe to delete).
*   **Accessibility**: High contrast ratios, support for large fonts, and screen reader compatibility.

## 2. Design Tokens
*   **Typography**: Using `GoogleFonts.cairo` (optimized for Arabic and English).
*   **Colors**: Dynamic color scheme that adapts to light/dark mode and merchant branding.
*   **Spacing**: 8dp grid system.

## 3. Atomic Design
*   **Atoms**: Base buttons, input fields, icons.
*   **Molecules**: Search bar, Product card, Rating star row.
*   **Organisms**: Product grid, Navigation bar, Live stream overlay.
*   **Templates**: Product detail page layout, Checkout flow.
