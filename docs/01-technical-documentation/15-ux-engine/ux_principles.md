# UX Principles: Zero Cognitive Load (ZCL)

## 1. Interaction Rules
*   **One Action per Screen**: Minimize decision paralysis.
*   **Contextual Assistance**: Information tooltips appear only when relevant.
*   **Skeletal Loading**: Never show a blank screen; use shimmer effects to show the structure of content while it's loading.

## 2. Visual Hierarchy
*   **F-Pattern Scanning**: Critical info (Price, Buy button) placed along common eye paths.
*   **Color as Feedback**: Green for success/money, Red for warnings/errors, Blue for information.
*   **Typography Scale**: Clear distinction between headings, body, and meta-data (specs).

## 3. Empty States & Feedback
*   "No results found" should always suggest an alternative or the "Request a Product" feature.
*   Tactile feedback (haptics) for successful actions on mobile.
