# Business Intelligence (BI) & Data Analytics

## 1. Data Warehousing
*   **ETL Process**: Extract, Transform, Load data from production DBs to an analytical store (e.g., BigQuery, ClickHouse).
*   **Anonymization**: PII data (Names, Phones) is hashed before entering the analytics layer to ensure privacy.

## 2. Key Performance Indicators (KPIs)
*   **Customer Lifetime Value (CLV)**: Predicting how much a user will spend over a year.
*   **Churn Rate**: Identifying users who haven't opened the app in 30 days.
*   **Heatmaps**: Using tools like PostHog or Firebase to see where users struggle in the UI.

## 3. Reporting
*   Automated daily emails for store owners summarizing their sales and most-viewed products.
*   Admin dashboard with real-time geographic distribution of orders.
