# Mahmoud Sovereign Mobile Ecosystem (Flutter & Native)

Enterprise-grade multi-vendor mobile application designed for Syria and the MENA region.

## 🛡️ Security Mastery (Native-First)
Unlike standard hybrid apps, all critical security decisions are handled in **Kotlin (Android)** and **Swift (iOS)** layers.
- **Environment Integrity**: Triple-algorithm root/jailbreak detection.
- **Anti-Tamper**: Protection against components like Lucky Patcher and Xposed.
- **Installer Validation**: Blocks sideloaded or modified APKs by verifying the Play Store/App Store signature.
- **Code Obfuscation**: Fully hardened via ProGuard/R8.

## 🚀 Key Features
- **Immersive Catalog**: High-performance product discovery with AR (Augmented Reality) integration.
- **Reverse Bidding**: Consumer-led marketplace where merchants compete for orders.
- **Sovereign Wallet**: Secure digital ledger with immutable forensic logging.
- **Real-time Tracking**: Map-integrated logistics for precise delivery monitoring.
- **Live Commerce**: Mobile stream studio for merchants to engage audiences live.

## 🛠️ Technical Stack
- **Framework**: Flutter 3.x (Presentation Only)
- **Native**: Kotlin 1.9+ / Swift 5.10+ (Decision Engine)
- **State Management**: BLoC / Cubit
- **Networking**: Dio with Native Interceptors
- **Typography**: Cairo (Arabic) & Roboto (English)

## ✅ Quality Standards
- **ZCL (Zero Cognitive Load)**: UI designed for maximum clarity and minimum mental effort.
- **Global Compliance**: W3C accessibility and RTL standards verified.
