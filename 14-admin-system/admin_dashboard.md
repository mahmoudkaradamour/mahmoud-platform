# Admin Control & Governance

## 1. Global Visibility
*   **KPIs**: GMV (Gross Merchandise Value), Total Users, Active Streams, Logistics health.
*   **Merchant Audit**: Review new merchant applications, verify legal documents.
*   **Financials**: Platform commission tracking, settlement approval.

## 2. Moderation Tools
*   **Content**: Remove products or ban sellers violating policies.
*   **Disputes**: Admin panel to resolve buyer/seller complaints with chat history and order details.

## 3. Configuration
*   **Invoice Designer**: Drag-and-drop tool to customize QR-coded invoice templates.
*   **Platform Settings**: Commission rates, supported currencies, global promotions.
