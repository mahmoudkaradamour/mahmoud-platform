# Social Media Sync & Integration

## 1. WhatsApp Business API
*   **Auto-Catalog Sync**: New products in the platform are automatically pushed to the merchant's WhatsApp Business Catalog.
*   **Order Notifications**: Sending order confirmations and tracking links via WhatsApp.

## 2. Meta (Facebook & Instagram)
*   **FB Shop Sync**: Integration with Meta Commerce Manager.
*   **Auto-Posting**: Option for sellers to auto-post "New Arrival" or "Promotion" content to their FB Page/IG Profile when published in the app.

## 3. Webhooks & API Keys
*   Each seller provides their own Meta/WhatsApp API keys in the dashboard.
*   Encrypted storage for external tokens.
